# Proxy-Switcher.sh

Proxy-Switcher is a simple `bash` script to activate or deactivate any HTTP/S proxy configuration.

## Usage

- To set up a proxy: `$ source proxy-switcher.sh http(s)://host:port`
- To remove a proxy: `$ source proxy-switcher.sh`
